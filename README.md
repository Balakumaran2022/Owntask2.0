# ownTask 2.0 — Enterprise Task Command Center

Built by **iEyal Solutions**, ownTask gives growing business teams Projects, SLAs, recurring automation, approval workflows, and real-time analytics in one unified, high-performance platform.

## 🚀 Key Capabilities

- **3-Level Hierarchy**: Organize work cleanly across Workspaces, Projects, Subjects, and Tasks.
- **Dual Board Types**: Seamlessly switch between Task Boards (Kanban) and Ticket Boards (SLA-driven queue management).
- **Two-Level SLA Enforcement**: Configurable countdown timers and breach thresholds at both project and individual task levels.
- **Immutability & Audit Trail**: 21 traceable system events logged permanently for compliance and team accountability.
- **Interactive Mock UI Elements**: Built with 100% pure HTML and CSS for lightning-fast rendering.
- **Dynamic SEO Engine**: Full meta descriptions and Open Graph cards tailored dynamically across pages.

## 🛠️ Technology Stack

- **Frontend Core**: React 18 + Vite 8
- **Styling**: Vanilla CSS (`index.css`) with curated modern tokens, glassmorphism, and responsive design
- **Animations**: Framer Motion & CSS keyframes

## ⚡ Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
