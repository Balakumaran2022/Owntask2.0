Enterprise Structural Hierarchy

Built from the ground up to support strict multi-level governance: Organisation → TaskProject → TaskSubject → Task → Subtask.

Auto-Generated Readable IDs

Every task receives an auto-generated alphanumeric ID formatted as TSK-XXXXXXXX. Instantly reference tasks across Slack, email, and Git pull requests.

Dual Board Architecture

TaskProjects support both task_board (traditional Kanban) and ticket_board (automated escalation rules, SLA breach alerts, and role-scoped triage queues).

Round-Robin Assignment

Assign tasks directly to an individual (assignedTo) or target a team (teamId) to automatically distribute workload evenly across active members.

Granular Task Capabilities

Every core task field engineered for transparency and operational scale.

SLA Enforcement Engine

Two-level SLA tracking covering absolute deadlines (slaDeadline) and status duration (targetDuration per status). Real-time tracking of slaBreached, slaBreachedAt, and slaBreachedBy.

Status SLA: In Progress 4h Target

Recurring Automation Series

Automate daily, weekly, monthly, yearly, or custom recurrence schedules. Features smart flags like skipWeekends and skipHolidays, with auto-calculated nextDueDate and dynamic offset formulas.

Approval Gate Workflows

Configure project-level approval gates where requiresApproval: true halts status transitions until signed off. Explicitly tracks approvalRequestedBy, approvedBy, and timestamps.

Approved Status Gate

21 Immutable Audit Events

Every mutation is permanently logged with IP address, performer, and full oldValue → newValue diffs. Covers STATUS_CHANGE, DUE_DATE_CHANGE, BULK_UPDATE, and 18 more events.

Immutable Log

Subtask Governance Rules

Subtasks share the exact same robust model as parent tasks. Live counters (subtaskCount / completedSubtaskCount) keep progress visible. Choose manual completion or all auto-closure rules.

Pre-Built Task Library

Install enterprise subject templates with one click: Bug Tracker, Feature Backlog, Customer Onboarding, HR Requests, and IT Support Tickets. Instantly deploys custom fields, statuses, and SLAs.
